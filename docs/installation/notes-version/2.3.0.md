# Version 2.3.0

Cette version se concentre sur l'amélioration des performances et la mise à jour de composants techniques.

Elle apporte également [un exemple de déploiement "tout-en-un"](../README.md) (initié en 2.2.9) pour faciliter la mise
en place de la solution.

## Principales modifications

### Changements bloquants

> [!CAUTION]
> La variable d'environnement APP_RUNTIME utilisée dans les versions antérieures est obsolète et doit être supprimée!

### Changements mineurs

* [frontend] Montées de version vers React 19 et antd 6
* [frontend] Ajout de la variable d'environnement REACT_API_API_PREFIX pour les installations sur un seul domaine
* [backend] Montée de version vers symfony 8
* [backend] Passage des tests backend sous PHPUnit
* Diverses corrections / améliorations (voir changelog)

## Migration depuis une version 2.2

La seule différence notable en termes de déploiement est la suppression de la variable d'environnement APP_RUNTIME.

La version 2.3.0 peut être déployée comme les versions précédentes, ou bien utiliser le déploiement "tout-en-un"
(voir [README](../README.md)), qui pourra également [simplifier les futures montées de version](../maj.md).
